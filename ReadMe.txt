Please use Visual Studio 2010 or above to run this project.

To run the application
* Enter code name
* Enter blood color 
* Enter no of antennas
* Enter no of legs
* Enter home planet

The entered details will be displayed.

Then Press 1 to export as Plain text.
Press 2 to export as PDF.


The saved location will be in the
-->  Alien_on_earth\bin\PDF & Txt\
inside your project folder
